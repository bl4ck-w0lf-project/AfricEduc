#  AfricEduc

> **La plateforme web qui simplifie la gestion des collèges africains.**

AfricEduc est une application web de gestion scolaire conçue pour accompagner les collèges dans leur transformation numérique. Elle centralise l'ensemble des opérations académiques, administratives et financières au sein d'une interface moderne, intuitive et performante.

---

#  Présentation

La gestion d'un établissement scolaire implique de nombreuses tâches quotidiennes : suivi des élèves, gestion des enseignants, calcul des moyennes, génération des bulletins, gestion des paiements, organisation des classes, etc.

AfricEduc a été développé pour automatiser ces processus et offrir aux établissements un outil simple, rapide et fiable.

L'objectif est de permettre aux équipes administratives de gagner du temps afin de se concentrer sur l'essentiel : la réussite des élèves.

#  Objectifs du projet

AfricEduc a pour ambition de :

- Digitaliser la gestion des collèges africains.
- Réduire les tâches administratives répétitives.
- Automatiser les calculs de moyennes.
- Faciliter la génération des bulletins.
- Améliorer le suivi des élèves.
- Simplifier la gestion des paiements.
- Offrir une plateforme moderne, rapide et évolutive.

# Installation

### 1. Cloner le dépôt

```bash
git clone https://github.com/votre-utilisateur/AfricEduc.git
```

### 2. Accéder au projet

```bash
cd AfricEduc
```

### 3. Configurer la base de données

- Créer une base de données MySQL.
- Importer le fichier SQL du projet.
- Modifier les paramètres de connexion dans le fichier de configuration.

### 4. Lancer le serveur

Vous pouvez utiliser :

- XAMPP
- Laragon
- WAMP
- MAMP
